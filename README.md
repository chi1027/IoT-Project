# IoT-Project
Smart Garden

## Objective:
- 實現全自動智慧植物系統,讓使用者不用額外耗費心力。

## Sensors and Actuators Used:
- Soil Moisture Sensor
- Light Sensor
- I2C LCD
- DHT22
- Buzzer
- RGB LED
- Raindrop Sensor
- 4 Channel Capacitive Touch Module
- Dust Sensor
Extra:
- Arduino
- Siri Control
- Line Notify
- Light system
  - 12V 5050 12SMD LED White Light
  - Relay Module
  - Transformer
  - Plug
- Watering system
  - Mini Submersible Water Pump
  - Water Pipe
  - Relay Module
  - Bottle with Water
